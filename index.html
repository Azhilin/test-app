AI Adoption Metrics Report

Jira Cloud sprint data · HTML & Markdown output

Generates an AI adoption metrics trend report from your Jira sprint data. The report tracks velocity, story-point completion, ticket throughput, and AI-assisted issue ratios across your configured sprint history. Output is written to reports/<timestamp>/ as both report.html and report.md.

Run Report Generation
Invokes python main.py on the server and streams the output below. Configure credentials in the Jira Connection tab before running.
